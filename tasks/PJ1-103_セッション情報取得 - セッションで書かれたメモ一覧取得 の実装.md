いいところ攻めてます。
IndexedDB 用の Repository に {{getSessionById}} */* {{getMemosBySessionId}} *を追加したサンプル*を書きます。

前に出した {{createSession / completeSession / saveMemo}} と同じスタイルで、
*シンプルに「getAllして filter」する実装*にしておきます（必要なら後から index 最適化）。

----

## 1️⃣ {{sessionsRepo.ts}} に {{getSessionById}} を追加

### 📄 {{src/lib/db/sessionsRepo.ts}}（サンプル）

```import { SessionRecord } from '@/types/session';
import { getDB } from './openDB';

const SESSION_STORE = 'sessions';

function generateId() {
  if (typeof crypto !== 'undefined' && 'randomUUID' in crypto) {
    return crypto.randomUUID();
  }
  return `session-${Date.now()}-${Math.random().toString(16).slice(2)}`;
}

/** テーマID一覧から新しいセッションを作成して保存 */
export async function createSession(themeIds: string[]): Promise<SessionRecord> {
  const db = await getDB();
  const now = new Date().toISOString();

  const session: SessionRecord = {
    id: generateId(),
    themeIds,
    startedAt: now,
    endedAt: null,
    memoCount: 0,
  };

  const tx = db.transaction(SESSION_STORE, 'readwrite');
  await tx.store.add(session);
  await tx.done;

  return session;
}

/** セッションを終了状態に更新（memoCount も更新） */
export async function completeSession(
  sessionId: string,
  memoCount: number,
): Promise<void> {
  const db = await getDB();
  const tx = db.transaction(SESSION_STORE, 'readwrite');
  const store = tx.store;

  const existing = (await store.get(sessionId)) as SessionRecord | undefined;
  if (!existing) {
    console.warn('completeSession: session not found', sessionId);
    await tx.done;
    return;
  }

  const updated: SessionRecord = {
    ...existing,
    memoCount,
    endedAt: new Date().toISOString(),
  };

  await store.put(updated);
  await tx.done;
}

/** 全セッション取得（履歴一覧用） */
export async function getAllSessions(): Promise<SessionRecord[]> {
  const db = await getDB();
  const tx = db.transaction(SESSION_STORE, 'readonly');
  const all = await tx.store.getAll();
  await tx.done;
  return (all ?? []) as SessionRecord[];
}

/** ID から単一セッションを取得（履歴詳細 & 完了画面用） */
export async function getSessionById(
  sessionId: string,
): Promise<SessionRecord | undefined> {
  const db = await getDB();
  const tx = db.transaction(SESSION_STORE, 'readonly');
  const record = (await tx.store.get(sessionId)) as SessionRecord | undefined;
  await tx.done;
  return record;
}
```

{quote}✅ ポイント

- {{getSessionById}} は *主キー lookup* なので {{store.get(id)}} だけでOK
- 履歴詳細 {{/history/[id]}} や {{/session/complete?sessionId=...}} でそのまま使えます{quote}

## 2️⃣ {{memosRepo.ts}} に {{getMemosBySessionId}} を追加

### 📄 {{src/lib/db/memosRepo.ts}}（サンプル）

```import { MemoRecord } from '@/types/memo';
import { getDB } from './openDB';

const MEMO_STORE = 'memos';

function generateId() {
  if (typeof crypto !== 'undefined' && 'randomUUID' in crypto) {
    return crypto.randomUUID();
  }
  return `memo-${Date.now()}-${Math.random().toString(16).slice(2)}`;
}

/**
 * メモを保存（同じ id があれば上書き）
 * 呼び出し側で id を決めてもいいし、任せてもOKなようにしている。
 */
export async function saveMemo(
  memo: Omit<MemoRecord, 'id' | 'createdAt' | 'updatedAt'> & {
    id?: string;
  },
): Promise<MemoRecord> {
  const db = await getDB();
  const now = new Date().toISOString();

  const record: MemoRecord = {
    id: memo.id ?? generateId(),
    createdAt: now,
    updatedAt: now,
    ...memo,
  };

  const tx = db.transaction(MEMO_STORE, 'readwrite');
  await tx.store.put(record);
  await tx.done;

  return record;
}

/**
 * 指定セッションに紐づくメモをすべて取得する
 * - セッション詳細画面 `/history/[id]` 用
 * - order 昇順に並び替えて返す
 */
export async function getMemosBySessionId(
  sessionId: string,
): Promise<MemoRecord[]> {
  const db = await getDB();
  const tx = db.transaction(MEMO_STORE, 'readonly');
  const store = tx.store;

  // MVP: index なしで getAll → filter
  const all = (await store.getAll()) as MemoRecord[];
  await tx.done;

  const filtered = (all ?? []).filter(memo => memo.sessionId === sessionId);
  // テーマ順で並べたいので order 昇順にソート
  filtered.sort((a, b) => a.order - b.order);

  return filtered;
}
```

{quote}🔎 最適化余地メモ

- {{openDB}} で {{memos}} に {{sessionId}} の index を貼っておけば
{{store.index('by_sessionId').getAll(sessionId)}} で高速に取れます
- まずは *実装シンプル優先で getAll → filter* にしておいて、
パフォーマンスが気になってきたら index 版に差し替えでよさそうです{quote}

## 3️⃣ {{/history/[id]}} や {{/session/complete}} での利用イメージ

ざっくり、ページ側からはこんな感じで使えます：

```import { getSessionById } from '@/lib/db/sessionsRepo';
import { getMemosBySessionId } from '@/lib/db/memosRepo';

async function loadSessionDetail(sessionId: string) {
  const session = await getSessionById(sessionId);
  if (!session) {
    throw new Error('Session not found');
  }

  const memos = await getMemosBySessionId(sessionId);

  return { session, memos };
}
```

----

これで：

- 履歴一覧 {{/history}} → {{getAllSessions}}
- 履歴詳細 {{/history/[id]}} → {{getSessionById}} + {{getMemosBySessionId}}
- 完了画面 {{/session/complete}} → {{getSessionById}}（＋必要なら memos）

という流れが作りやすくなりました。
