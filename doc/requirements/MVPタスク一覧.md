以下は、**MVP に必要な機能**に対して、**「実装タスク」を体系的にすべて洗い出したリスト**です。
設計 → UI → 実装 → データ → テスト → デプロイ まで一本の動線になるよう構成しています。

# ✅ MVP タスク一覧（1分メモアプリ・Web 版 / IndexedDB 保存・テキスト＋手書き対応）

---

# 1. 要件確定タスク（最初にやるべきこと）

## 1-1. 仕様確定

- [ ] MVP で使うテーマ数を「固定 10」に確定する
- [ ] 入力時間を「固定 60 秒」に確定する
- [ ] テキストと手書きキャンバスの形式（Canvas サイズ・画像形式）を決定する
- [ ] セッション中の画面遷移方式（自動遷移 / インターバルなし）を決める
- [ ] IndexedDB のスキーマ確定
- [ ] 最低限の画面一覧を確定（後述）

# 🎉 完成イメージ（MVP はここまでカバーできれば十分）

- テーマ 10 個がランダムに選ばれる
- 1 テーマ 1 分でテキスト＋手書き入力できる
- 入力結果は IndexedDB に保存
- 履歴の一覧・詳細が確認できる
- テーマ管理で ON/OFF ができる

これで「個人が毎日使える1分メモアプリ」として成立します。
